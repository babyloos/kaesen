# jpy2btcRobot

## Description
日本国内のBitcoin交換所で日本円取引をするためのRubyのクライアントライブラリです．

This is the Ruby Client Library that exchange Bitcoin to JPY at Exchange Market in Japan.

## Support Market

### Now suppported markets

  + Bitflyer
  + Btcbox
  + Coinckeck
  + Zaif

### Not yet
  + LakeBTC
  + Kraken
